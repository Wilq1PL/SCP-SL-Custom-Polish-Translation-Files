Jak pobrać i używać:

Numero Uno - pobierz ZIPa i rozpakuj go w folderze z tŁumaczeinami gry:

Steam\steamapps\common\SCP Secret Laboratory\Translations.

Aby znaleźć folder przejdź do Biblioteki Steam, kliknij PPM w SCP:SL, wejź w zakadkę Zarządzaj i wybierz Przeglądaj pliki lokalne, i woejdź do folderu Translations

Następnie otwórz SCP:SL i w ustawieniach zmień język na: Polski (Custom by WilqPL)
